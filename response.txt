 Unterscheidung between schema and ldif files
The difference between a schema and an LDIF (Lightweight Directory Access Protocol) file is that a schema defines the structure of an LDAP directory, while an LDIF file contains the actual data for that directory. In other words, a schema outlines the organization of the directory, while an LDIF file provides the details of the entries within that directory.
